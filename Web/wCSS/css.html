属性 分类 解释 例子
引用方式 行内样式 在元素标签添加,权重最高
嵌入式样式 在head内添加style标签 style>样式设置style>
外部链接 在head添加link标签,并引入外部文件 link rel="stylesheet" href="style.css">
导入式 在head添加style的@import导入css文件 style type="text/css" > @import"css文件的地址" /style>
link和import的区别 link是html加载前就引用,import是html加载后才引用
  • 若使用Javascript动态添加引用CSS,需要使用link引用方式才可以实现
  • link最常用且多文件引动同一个CSS文件,css只需要下载一次
选择器 通配符选择器 * *{}
标签选择器 元素{} p{}
类选择器 .类名 .one{}
ID选择器 #id名{} #one{}
紧跟相邻兄弟选择器 element+element { 用于选取第一个指定的元素之后（不是内部）紧跟的元素。} div+p{ 选择 div 元素之后紧跟的每个p 元素，并设置其背景色：}
非紧跟兄弟元素 element1~element2 选择器 element1 之后出现的所有 element2。 两种元素必须拥有相同的父元素， 但是 element2 不必直接紧随 element1。 所有相同的父元素中位于 p 元素之后的所有 ul 元素设置背景：p~ul
多元素选择器 选择多个元素{} p,div,ul{设置样式}
属性选择器(对带有指定属性的HTML元素设置样式) 元素[属性] 选择带有指定属性的元素 a[targe]{}
元素[属性="属性值"] 某个元素的属性是某个值的 [target="_blank"]{选择target="_blank"}的所有元素
元素[属性~="value值"] 选择属性包含单词"value"的元素 [title~="flower"]{选择title属性包含单词"flower"的所有元素}
元素[属性^="value值"] 选择属性值以"value"开头的所有元素 a[src="^https"]{选择其src属性值以https开头的每个a元素}
元素[属性$="value值"] 选择属性值以value结尾的所有元素 a[src$=".pdf"]{选择其src属性以.pdf结尾的所有a元素}
元素[属性*="value"] 选择属性值包含"value"的所有元素 a[src*="abc"]{选择其src属性中包含sbc子串的每个a元素}
元素[属性|="value"] 选择属性值为"value"或以"value"开头的所有元素 [lang|=en]{选择lang属性值以""en"开头的所有元素}
伪类选择器 动态伪类:这些伪类并不存在HTML中,只有当用户和网站交互的时候才能体现出来
锚点伪类 :link a:link选择所有未被访问的链接
:visited a:visited选择所有已被访问的链接
用户行为伪类 :hover a:hover选择鼠标指针位于其上的链接
:active :active选择活动链接
:focus input:focus选择获得焦点的input元素
UI元素状态伪类
:enabled input:enabled选择每个启用的input元素
:disabled input:disabled选择每个禁用的input元素
:checked 选择每个被选中的input元素
CSS3结构类 元素:nth选择器:选择父元素底下的第N个元素选择的元素

:first-child,:last-child,:nth-child(),:nth-last-child(),:nth-of-type(),:nth-last-of-type(),:first-of-type(),:last-of-type(),:only-child(),:only-of-type(),:empty

元素:first-child{} 选择属于其父元素的首个子元素的每个Element元素 p:first-child{span class="importantNews">父元素地下的第一个必须是P元素,否则不能成功 }
元素:last-child{} 选择属于其父元素的最后一个子元素的每个Element元素 p:first-child{ 父元素地下的最后一个必须是P元素,否则不能成功 }
元素:nth-child(N) 选择属于其元素的第N个子元素,不论元素的类型

N代表数字:选择某元素下的第number个元素[1开始]

简单的表达式:取值从0开始,比如:2n

元素:nth-child(odd){匹配奇数的元素}

元素:nth-child(even){匹配偶数的元素}
元素:nth-last-child(n) 倒着数进行从最后一个元素算起 方法同上
元素:nth-of-type(n) 选择属于其父元素的第N个元素,必须和元素相同 p:nth-of-type(n){p元素的父元素下的第n个p元素}
元素:first-of-type/元素:last-of-type 选择匹配属于其元素的特定类型的首个子元素 p:first-of-type{}/p:last-of-type{}
元素:nth-last-of-type{} 选择属于父元素的特定类型的第N个子元素的每个元素,从最后一个元素开始计数
元素:only-child 匹配属于其元素的唯一子元素的每一格元素 p:only-child{选择属于其元素唯一的子元素的每个p元素}
元素:only-of-type 匹配属于其元素的特定类型的唯一子元素的每个元素 p:only-of-type{选择属于其父元素唯一的p元素}
元素:empty 匹配没有子元素(包括文本节点)的每个元素 p:empty{选择没有子元素的每个P元素(包括文本节点)}
否定选择器:not 元素:not(子元素/子选择器),匹配非指定元素/选择器的每个元素 :not(p)选择非p元素的每个子元素
伪元素 元素:first-line 对元素的第一行文本进行格式化,只能用于块级元素 p:first-line{}
元素:first-letter 用于向文本的首字母设置特殊的样式 p:first-letter{}
元素:before 在元素的内容前面插入新的内容,常用"content"配合使用 p:before{content:"内容"}
元素:after 在元素的内容后面插入新内容 p:after{content:""}
元素::selection 用于设置在浏览器中选中的文本后的背景色与前景色 input::selection选择被用户选取的元素部分
字体属性 font属性简写 font:font-style font-variant font-weight font-size/line-height font-family
字体样式 font-style:normal/italic/oblique 字体变形:设置元素中文本大小写字母 font-variant:normal/small-cap
字体粗细 font-weight:normal/bold/bolder/lighter/100-900 字体大小 font-size:相对单位/绝对单位
字体类型 font-family:字体名/字体集 字体的颜色 color:颜色名/十六进制/RGBA

行高 line-height:长度值/百度值
文本属性 设置元素单词之间间距 word-spacing:数值 设置字母之间的间距 letter-spacing
设置元素内文本的大小写 text-transform:capitalize/uppercase/lowercase/none 设置元素内文本的装饰 text-decoration:underline/overline/line-through/blink/none
盒模型 概念 1.盒模型是存放网页中的各种元素 2.网页设计中内容,如文字,图片等元素,都是盒子(DIV嵌套)
盒模型的宽和高 width/height:长度值/百分比/auto max-width/max-height:长度值/百分比/auto min-width/min-height:长度值/百分比/auto
